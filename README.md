## Commands
All commands use `/r` prefix:
- `/r play <song>` - Play music
- `/r skip` - Skip current track
- `/r queue` - View queue
- `/r mystats` - Your listening stats
- `/r leaderboard` - Server leaderboard
- `/r help` - All commands
